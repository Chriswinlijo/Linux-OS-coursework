## Overview
This technical journal records the setup, security hardening, monitoring, and performance testing of a **headless Ubuntu Server VM**. The server is managed remotely using **SSH** from a separate workstation, with all administration completed through the command line.

Key areas covered:
- Command-line administration and evidence
- Secure remote access and baseline hardening
- Workload-based performance measurement
- Weekly reflection and continuous improvement
